STEP 1: TEST LOCALLY
-------------------
1. Open terminal/command prompt
2. Go to your project folder: cd /path/to/GOPROMO-IMEI
3. Install: npm install
4. Test: npm run dev
5. Open http://localhost:3000 in browser
6. Test all features, then press Ctrl+C to stop

STEP 2: PUSH TO GITHUB
---------------------
1. Go to github.com, create new repository named "GOPROMO-IMEI"
2. In terminal:
   git init
   git add .
   git commit -m "Ready for deployment"
   git remote add origin https://github.com/Bangblonk/GOPROMO-IMEI.git
   git push -u origin main
3. When asked for password, use GitHub Personal Access Token (not your password)

STEP 3: DEPLOY TO VERCEL (EASIEST)
---------------------------------
1. Go to vercel.com
2. Sign up with GitHub
3. Click "New Project"
4. Select "GOPROMO-IMEI" repository
5. Click "Deploy"
6. Wait 2-3 minutes
